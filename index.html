System Architecture
01

Qualify Enquiries Using AI

I use n8n's multi-step forms to build clean, conversational booking experiences instead of long, boring forms. In this setup, when a client asks for a meeting, an AI text classifier screens their request first.

n8n-nodes-base / qualify-enquiries
02

Split Form For Better User Experience

I broke the onboarding flow into smaller sections to keep users engaged and avoid overwhelming them. Once the user agrees to the terms, the workflow advances to the next step to let them choose an available date and time.

n8n-nodes-base / split-forms
03

Send Acknowledgement and Start Approval

As soon as the user finishes the forms, the system sends them a beautiful email confirmation instantly. Simultaneously, the main workflow triggers a hidden background path that alerts me of the request.

n8n-nodes-base / background-trigger
04

Approve or Decline Appointment

I built a 'human-in-the-loop' approval flow using an n8n automation linked directly to my inbox. The system pauses and sends me a clean email containing the AI's summary and two simple action buttons: Confirm or Decline.

n8n-nodes-base / google-calendar-sync
